mylogreg.py 
===========

Perform logistic regression to predict whether a student gets admitted into a university, based on the applicant’s scores on two exams. Data set corresponds to Part 1 of Problem Set #2 of Andrew Ng’s Machine Learning Class.


mylogre_reg.py 
==============

Performs regularized logistic regression. Aim is to predict whether a microchip from a fabrication plant will pass quality assurance, based on the results of two tests. Data set corresponds to Part 2 of Problem Set #2 of Andrew Ng’s Machine Learning Class.
